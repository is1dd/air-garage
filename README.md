# Air Garage Clone

AirGarage enables churches and businesses with extra spaces to rent them out to drivers on-demand and boost their bottom line.

## Tech Stack

**Client:** HTml, Css & Java-Script

## Features

- Log in.
- Sign up .
- Find Parking.
- Selecting Parking Slots.
